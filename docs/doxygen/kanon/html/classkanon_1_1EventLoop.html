Detailed Description

A loop that process various events.

There three phases in loop: (1)select/poll/epoll(event dispatching) -> (2)
(2)handle events(including timerfd) -> (3)
(3)functors -> (1)
They constrcut a loop to accept event and handle them, and also handle the functors that user pushs.
